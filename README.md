# ⚽ FutballoveKits (Backend)

This is the **backend** of the FutballoveKits eCommerce website — built using **Node.js, Express.js, and MongoDB**.  
It provides RESTful APIs for authentication, product management, and order/cart handling.

---

## 🚀 Features

- 🧑‍💻 **Admin Panel Support** – Add new products with name, image, and price  
- 🔐 **JWT Authentication** – Secure login and route protection  
- 📦 **Product Management** – Create, read, update, and delete football kits  
- 🛒 **Cart Management** (connected with frontend)  
- 🗄️ **MongoDB + Mongoose** – Database for storing users and products

---

## 🧰 Tech Stack

- **Node.js**
- **Express.js**
- **MongoDB & Mongoose**
- **jsonwebtoken**
- **dotenv**
- **cors**
- **bcryptjs**
